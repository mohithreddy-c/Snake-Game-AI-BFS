# 🐍 Snake AI using BFS

A Snake Game built using HTML, CSS and JavaScript.

## Features

- AI Mode (BFS Pathfinding)
- Manual Mode
- Wall Wrapping
- Self Collision Detection
- High Score Leaderboard
- Path Visualization
- Cyberpunk UI

## DSA Concepts Used

- Graphs
- Breadth First Search (BFS)
- Queue
- Path Reconstruction
- Hash Set

## Controls

- A → AI Mode
- M → Manual Mode
- R → Reset High Score
- Arrow Keys → Manual Control

## Technologies

- HTML
- CSS
- JavaScript
